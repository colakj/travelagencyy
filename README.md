## Local Setup

#### 1. Clone repo on your machine
  ```bash
  git clone https://github.com/colakj/travelagency.git
```

#### 2. Create docker containers
Move to the project directory and run the following command in the terminal:
  ```bash
  docker-compose up -d
```
